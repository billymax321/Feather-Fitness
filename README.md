# Feather Fitness Website

Production-ready Next.js + Tailwind starter based on the refined Figma page screenshots.

## Pages
- `/services`
- `/booking`
- `/membership`
- `/events`
- `/about`

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Replace placeholder images
The current build uses reusable placeholder image components so the layout stays responsive. Replace `PlaceholderImage` blocks with `next/image` once you export final Figma assets.

## Deploy
Push this folder to GitHub, then import the repository into Vercel or Netlify.
